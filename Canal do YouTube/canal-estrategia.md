## 1. QUEM É O MATHEUS (identidade do canal)

**Posicionamento:** Dono de assessoria de marketing que documentou a jornada de automatizar a própria empresa com IA — e está mostrando o processo em tempo real.

**O que diferencia:**
- Não é um professor de IA genérico
- Não é um dev ensinando a programar
- É um empreendedor mostrando o que realmente usa no dia a dia para escalar uma empresa operando com o mínimo de pessoas possível
- Usa ferramentas reais, com clientes reais, com resultados reais

**Frase de posicionamento (rascunho):**
> "Estou construindo uma assessoria que roda com IA — e estou mostrando tudo."

## 2. PÚBLICO-ALVO

**Perfil principal:** Empreendedores digitais, donos de agências, social media e freelas que querem usar IA para escalar sem contratar mais gente.

**O que eles buscam:**
- Ver o que é possível fazer (não necessariamente aprender a fazer)
- Inspiração + prova social de que funciona
- Ferramentas reais com casos de uso reais
- Alguém que já está fazendo e mostrando os bastidores

**O que eles NÃO querem:**
- Tutorial longo e técnico demais
- Teoria sem resultado prático
- Conteúdo genérico de "use o ChatGPT para ser mais produtivo"

**Problema com "agência":** termo genérico, sem diferencial, não comunica a proposta de valor.

**O que a Assessoria Alpha faz:** usa o termo "assessoria" + posicionamento premium (alpha = primeiro, superior). Faz sentido porque assessoria tem connotação de mais próximo, estratégico, consultivo — diferente de agência que parece fábrica.

### 6.3 Regras de título que funcionam

1. **Especificidade bate generalidade** — "Criei 7 templates em 2 horas" > "Como usar IA para design"
2. **Resultado em primeiro lugar** — o título entrega o fim antes do começo
3. **Curiosidade com tensão** — "o que aconteceu" / "você não vai acreditar" / "mudou tudo"
4. **Número sempre performa** — listas, quantidades, tempo, dinheiro economizado
5. **Primeira pessoa** — "eu fiz", "minha assessoria", "meu workflow" cria identificação

## 9. FORMATO E ESTILO DOS VÍDEOS

### Duração ideal
- **10–20 minutos:** ideal para o nicho (retém audiência qualificada e gera watchtime)
- **Shorts (60s):** para clips de resultados, antes/depois, bastidores rápidos

### Estrutura recomendada
```
0:00–0:30  Hook — mostra o resultado final antes de explicar
0:30–2:00  Contexto — quem é você, qual foi o problema
2:00–15:00 Jornada — o processo, as decisões, os erros
15:00–18:00 Resultado — números, antes/depois, aprendizados
18:00–20:00 CTA — próximo vídeo, comunidade, comentário
```

### Tom de voz
- Honesto e direto (sem hype exagerado)
- Mostra o que não funcionou junto com o que funcionou
- Não tutorial passo a passo — mais "veja o que eu fiz e por quê"
- Linguagem de empreendedor, não de professor

### 12.2 Decodificando a fórmula de título

**Fórmula 1 — O "setor inesperado":**
```
"Estou [vendendo/automatizando] + [setor NÃO ÓBVIO] + com Claude Code + ([resultado concreto])"
```
- **A genialidade:** juntar Claude Code (tecnologia de ponta) com açougue, imobiliária (setores tradicionais) cria choque cognitivo
- "Carne com Claude Code?" → O espectador não consegue não clicar, porque não faz sentido à primeira vista
- O resultado em parênteses fecha a promessa: "toneladas", "200k" — específico, crível, impressionante

**Fórmula 2 — O resultado financeiro direto:**
```
"Como fiz R$[valor exato] com Claude Code em [contexto específico]"
```
- Dinheiro concreto + prazo específico = credibilidade máxima
- "em 1 call" é o detalhe que causa inveja produtiva ("ele fez isso num telefonema?")
- O espectador quer saber COMO — o título garante o clique

**Fórmula 3 — A construção em primeira pessoa:**
```
"Construí [coisa concreta] com Claude Code"
```
- Mais simples, mais direto
- Funciona melhor quando a "coisa" já impressiona por si só

---

### 12.3 Por que esses vídeos bombam — análise profunda

**1. Setor inesperado como isca de curiosidade**
O Vagner não fala "automatizei um processo de vendas com IA" (genérico). Ele fala "vendo CARNE com Claude Code". O cérebro humano estranha essa combinação e precisa resolver o paradoxo. Isso é o clique.

**2. Resultado é a promessa, não o processo**
O título não explica o que ele fez — diz o que ele conseguiu. O processo fica para dentro do vídeo. Isso preserva o mistério e garante o watchtime.

**3. Canal serve de funil para produto**
Cada vídeo é um case do MazyOS (produto dele). O YouTube não é o negócio — é a prova social que vende o produto. Essa clareza de objetivo muda tudo na estratégia de conteúdo.

**4. Personagem com posicionamento claro**
"Ganho dinheiro dormindo" é uma promessa de vida. O espectador quer aquilo, não só a ferramenta. Vagner vende um estilo de vida automatizado com IA, não aulas sobre Claude Code.

### 12.5 Fórmula de thumbnail CONFIRMADA (Vagner Mazzeo)

Confirmado pelo Matheus que assistiu os vídeos. Layout fixo em todos os vídeos que bombaram:

```
┌─────────────────────────────────────┐
│ [Logo Claude]          [💰 dinheiro] │
│  canto superior esq.                │
│                                     │
│         [imagem do setor]           │
│      (apartamento, carne, etc.)     │
│                                     │
│  TEXTO GRANDE COM O RESULTADO       │
└─────────────────────────────────────┘
```

| Elemento | Posição | Detalhe |
|---|---|---|
| Logo do Claude Code | Canto superior esquerdo | Sempre presente, branding da ferramenta |
| Dinheiro / moedas / R$ | Canto superior direito ou área de destaque | Sinaliza resultado financeiro |
| Imagem do setor | Centro / fundo | Apartamento nos de imóveis, carne (estilo Minecraft) nos de açougue |
| Texto de resultado | Destaque central | Grande, contrastante, específico |

**Por que funciona:**
- Logo do Claude = contexto imediato ("esse vídeo é sobre IA")
- Dinheiro = promessa de resultado ("vai me render algo")
- Imagem do setor = choque cognitivo ("IA + setor inesperado")
- Os três elementos juntos criam a pergunta no inconsciente: "como esses mundos se conectam?"

**Adaptação para o Vídeo 01 (Movisol):**

```
┌─────────────────────────────────────┐
│ [Logo Claude]          [☀️ + R$]    │
│                                     │
│      [painel solar ou sol]          │
│                                     │
│  "7 POSTS EM 1 DIA"                 │
│  ou "DO ZERO EM 1 CONVERSA"         │
└─────────────────────────────────────┘
```

- Logo Claude: canto sup. esq. (padrão)
- Elemento de dinheiro: símbolo R$ ou moedas ao lado de um sol/painel
- Imagem central: painel solar real ou ícone de sol estilizado
- Foto do Matheus: expressão de surpresa/orgulho à frente dos elementos
