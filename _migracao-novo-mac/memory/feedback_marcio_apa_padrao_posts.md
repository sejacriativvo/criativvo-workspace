# Posts do Marcio sempre usam a library de templates aprovada

Quando Matheus pedir um post novo do Marcio / A·P·A Previdenciário, **NÃO criar layout do zero**. Sempre puxar um dos 11 templates aprovados em [templates-posts-marcio-albrechete.html](Cliente-Criativvo/ativos/marcio-albrechete/redes-sociais/templates-posts-marcio-albrechete.html) como base.

**Why:** O sistema visual foi iterado várias vezes com feedback detalhado do Matheus (v1 a v6) e tá no padrão "jurídico premium editorial" da identidade A·P·A. Inventar layout novo quebra a consistência e desperdiça o trabalho de ajuste fino que já foi feito (proporções, espaçamentos, paleta, cabeçalho fixo, CTA, arcos concêntricos, etc).

**How to apply:**
1. Identificar qual template casa com o conteúdo do post pedido (mapeamento em [[project-marcio-apa-design-system]])
2. Copiar o bloco HTML/CSS daquele template
3. Substituir só o conteúdo (texto, número, imagem) mantendo estrutura, tipografia, paleta
4. Se nenhum template casar 100%, escolher o mais próximo e adaptar minimamente (não criar um 12º template sem alinhamento prévio)
5. Sempre respeitar regras OAB ([[feedback-marketing-juridico-oab]]): sem captação ostensiva, sem garantia de resultado, sem mercantilização, cases anonimizados

**Quando criar template novo:** só quando Matheus pedir explicitamente ou enviar referência visual nova. Caso contrário, **reusa**.
