Todo post da Movisol precisa de legenda + hashtags criados junto com o visual.

**Legenda:**
- Post único: legenda longa e encorpada, completa a informação do visual
- Carrossel: legenda curta, introdução ou conclusão com CTA
- Estrutura: abertura forte → desenvolvimento → CTA

**Hashtags:**
- Máximo 5 por post
- Sempre incluir #movisol e/ou #ibitinga
- Demais hashtags: pesquisar as com mais tração no nicho de energia solar na data do post
- Não usar lista fixa — verificar relevância a cada post

**Why:** regra definida pelo Matheus para padronizar a entrega completa de cada post.

**How to apply:** nunca entregar apenas o visual. O output de cada post é: visual (template) + legenda + hashtags.
